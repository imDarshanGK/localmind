# Screenshots

This folder contains UI screenshots for LocalMind.

## How to contribute screenshots
1. Run LocalMind locally
2. Take screenshots of the UI
3. Save them here with these exact filenames:
   - `chat-ui.png`
   - `upload-panel.png`
   - `plugins-panel.png`
   - `settings-panel.png`
   - `sidebar-sessions.png`
4. Open a PR!
